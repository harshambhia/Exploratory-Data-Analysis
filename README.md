# 🔬 EDA Agent

An automated **Exploratory Data Analysis** web app built with Streamlit.  
Upload any CSV / TXT / Excel file and instantly get:

- 📋 Dataset overview & statistical summary
- 📊 Interactive distribution plots (histograms, box plots)
- 🔗 Correlation heatmaps & scatter explorer
- 🗂️ Categorical frequency analysis
- 🕳️ Missing value heatmaps
- 💡 Auto-generated insights & recommendations

## 🚀 Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Deploy on Streamlit Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your repo → select `app.py` → Deploy
